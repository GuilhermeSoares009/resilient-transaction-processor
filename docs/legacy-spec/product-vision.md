# Product Vision

## Visao
Entregar um processador de transacoes resiliente que garanta consistencia de saldo, idempotencia e rastreabilidade completa.

## Objetivos
- Garantir operacoes ACID para creditos e debitos
- Evitar duplicidade com idempotency keys
- Publicar eventos confiaveis para sistemas downstream

## Nao objetivos
- Suporte a pagamentos internacionais
- Interface grafica completa

## Personas
- Engenheiro de plataforma financeira
- Analista de risco e auditoria
